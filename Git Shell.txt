github for windows

push to github:
1.Open in Git Shell
2.git add -f dir (提交打开的目录下面的所有文件)
2(0).git add --all (提交打开的目录下面的所有目录和文件)
3.git commit -m "备注" (提交新增的文件)
4.git commit -a -m "备注" (提交内容有变化的文件)
5.git push

clone from github:
1.open git shell
2.cd dir
3.git clone url

从服务器上更新到本地：
git checkout -- file or dir

强制更新到服务器：
git push --force
